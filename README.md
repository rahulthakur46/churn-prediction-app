A dark-themed, cyberpunk-style AI web app that predicts customer churn using Machine Learning.

🔗 **Live Demo:** [Click Here](https://churn-prediction-app-nsdwobfmh7kuyer5t4kxam.streamlit.app)

---

## 🚀 Features

- 🔮 Real-time churn prediction with probability score
- 📊 Interactive analytics dashboard with Plotly charts
- 🎯 Personalized retention recommendations
- 🗂️ Filterable customer data table
- ⚡ Cyberpunk dark UI with neon accents

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| Python | Core language |
| Streamlit | Web framework |

## 🏃 Run Locally

```bash
git clone https://github.com/rahulthakur46/churn-prediction-app.git
cd churn-prediction-app
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Dataset

- 300 customer records
- 20 features (demographics, services, billing)
- Binary target: Churn Yes/No
